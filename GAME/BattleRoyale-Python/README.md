# Battle Royale: Python Edition 🔥🛡️

A **multiplayer 2D battle royale** game where players fight until only one remains! This game features **online multiplayer**, different weapons, shrinking play zones, and real-time action! 🕹️💥

---

## 🚀 Features
- ✅ **Multiplayer support** (via sockets) 🎮
- ✅ **Smooth player movement** with `pygame` 🚶‍♂️
- ✅ **Weapons system** (shoot and eliminate enemies) 🔫
- ✅ **Shrinking play zones** to make battles intense 🔥
- ✅ **Server-client architecture** for online gameplay 🌐
- ✅ **Real-time synchronization** for all players 🔄
- ✅ **Leaderboard & stats tracking** 🏆

## 🛠️ Installation & Setup
### 🔹 **1. Clone the Repository**
```bash
git clone https://github.com/yourusername/BattleRoyale-Python.git
cd BattleRoyale-Python
```

### 🔹 **2. Install Dependencies**
```bash
pip install -r requirements.txt
```

### 🔹 **3. Start the Server** (Run this on one computer)
```bash
python server/server.py
```

### 🔹 **4. Start the Client** (Run this on multiple computers to play)
```bash
python client/game.py
```

## 🎮 How to Play
- Move with **Arrow Keys / WASD** 🎮
- Shoot with **Spacebar** 🔫
- Survive and eliminate enemies! The last player standing wins! 🏆
