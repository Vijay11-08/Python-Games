## 🔧 Installation & Setup 🖥️⚙️
1. 📥 Clone the repository:
   ```bash
   git clone https://github.com/your-repository-link
   ```
2. 📂 Navigate to the project folder:
   ```bash
   cd game_project_folder
   ```
3. 📦 Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. 🎮 Run the game:
   ```bash
   python main.py
   ```
